# chrome小恐龙 项目报告

数字逻辑课程  Group x  郝广博 桂晓琬

 

 

 

 

 

## 背景介绍

Chrome浏览器在没有网络连接的时候，会显示一个小恐龙，点击空格就可以玩这个小游戏，十分有趣，所以我们在这次的project中使用Verilog硬件描述语言来实现这样一个小恐龙游戏。

### 输入输出交互选择

输入: Sword 实验平台上的按钮、拨动开关

输出: VGA显示器，数码管
